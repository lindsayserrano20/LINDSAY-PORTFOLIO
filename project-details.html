Below is a breakdown of the backend architecture and database integration used for this project.

PHP 8.x PDO MySQL

Database Connection

Utilizing PDO for secure, prepared statements to prevent SQL Injection.

<?php
// db_connect.php
$host = 'localhost';
$db = 'gensoc_db';
$user = 'root';
$pass = '';

try {
    $pdo = new PDO("mysql:host=$host;dbname=$db;charset=utf8", $user, $pass);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ATTR_ERRMODE_EXCEPTION);
} catch (PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}
?>

Data Submission Logic

Handling form data and executing the insertion into the user_info table.

// submit.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = $_POST['full_name'];
    $identity = $_POST['gender_identity'];

    $sql = "INSERT INTO user_info (name, identity) VALUES (?, ?)";
    $stmt = $pdo->prepare($sql);
    $stmt->execute([$name, $identity]);

    echo "Record added successfully!";
}
