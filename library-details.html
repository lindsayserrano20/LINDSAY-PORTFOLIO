Below is a breakdown of the Object-Oriented Programming (OOP) and file inclusion logic used for this project.

PHP 8.x OOP INCLUDE

Class Structure (Book.php)

Defining the properties and methods for book objects.

<?php
class Book {
    public $title;
    public $author;

    public function __construct($title, $author) {
        $this->title = $title;
        $this->author = $author;
    }
}
?>

Logic Implementation

Using include to load the class and create new book instances.

// index.php
include 'Book.php';

$library = [
    new Book("PHP for Beginners", "Lindsay"),
    new Book("Data Structures", "Lindsay")
];

foreach ($library as $item) {
    echo $item->title;
}
